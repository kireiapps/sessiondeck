# 🃏 Session Deck

> Turn forgotten AI sessions into a living deck of knowledge.

Session Deck transforms old Hermes conversations into an interactive deck of cards. Instead of letting past sessions become clutter, each card becomes a choice that helps your AI evolve.

## Features

🔥 **Discard**  
Permanently delete obsolete sessions and remove unwanted context.

✨ **Distill**  
Extract reusable workflows from a conversation and convert them into Skills.

🧠 **Imprint**  
Save important decisions and long-term preferences directly into Memory.

👁️ **Inspect**  
Open any session and review its complete history before taking action.

⏭️ **Pass**  
Skip the current card and send it to the bottom of the deck.

---

## Why?

Most AI conversations are forgotten after they're finished.

Session Deck treats conversations as assets instead of archives. Every session can become reusable knowledge, permanent memory, or simply disappear when it no longer provides value.

Instead of collecting history, your AI continuously curates it.

---

## Installation

```bash
cp -r session-deck ~/.hermes/skills/automation/session-deck
```

## Usage

```bash
python3 scripts/session_deck.py pick
```

Start the game.

```
Draw a random card.

Discard.
Distill.
Imprint.
Inspect.
Pass.

Repeat.
```
